# FitnessOS Platform

The web application for FitnessOS: a secure, multi-tenant operating system for online fitness coaching.

## Stack

- Next.js with App Router
- TypeScript
- Tailwind CSS
- Supabase (Development and Staging environments are isolated)

## Prerequisites

- Node.js 20.9 or newer
- npm
- Git

## Local development

```bash
npm install
copy .env.example .env.local
npm run dev
